# KAMAY

> Del quechua: mano, habilidad, poder de crear

Marketplace de servicios profesionales. Conecta clientes con freelancers verificados.

## Stack

- **Frontend:** Next.js 14 (App Router) + Tailwind CSS
- **Backend:** NestJS + Prisma + PostgreSQL
- **Pagos:** Stripe Connect
- **Cache:** Redis
- **Storage:** Cloudflare R2

## Estructura

```
kamay/
├── frontend/ → Next.js app
└── backend/ → NestJS API
```
